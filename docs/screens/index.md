# 화면 명세 인덱스

| 화면 | 파일 | 설명 |
|------|------|------|
| 메인 대시보드 | [main-dashboard.md](./main-dashboard.md) | 활성 세션 실시간 모니터링, 최초 진입 화면 |
| 세션 목록 | [session-list.md](./session-list.md) | 종료 세션 조회 및 삭제 |

## 화면 네비게이션

```
ccmonit 실행
    │
    ▼
메인 대시보드 ──l──▶ 세션 목록
    ▲                   │
    └───────ESC─────────┘
```

## 확장 계획

- 세션 목록 외에 다른 화면이 추가될 수 있다.
- 화면 간 네비게이션 구조도 함께 확장될 예정이다.
