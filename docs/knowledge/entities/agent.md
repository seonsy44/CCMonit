# Agent

## 정의
Agent는 session 내부에서 특정 역할을 맡아 실행되는 작업 주체다.

## 구분
- main agent
- subagent
- team 하위 agent

## 관찰 포인트
- 현재 상태
- 수행 중 task 수
- 마지막 활동 시각
- 누적 tool 호출
- 토큰 소모량
