# Alert

## 정의
Alert는 운영자가 빠르게 개입해야 할 이상 징후를 구조화한 이벤트다.

## 예시
- stuck
- token spike
- adapter degraded
- repeated error
- orphan ratio high

## 운영 원칙
경고는 단순히 많이 띄우는 것이 아니라, **해석 가능한 개입 포인트**를 제공해야 한다.
