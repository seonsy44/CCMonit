# Tool Usage

## 정의
Tool Usage는 특정 skill 또는 task 안에서 실제 도구 호출이 일어난 기록이다.

## 왜 중요한가
- 비용 원인을 가장 구체적으로 볼 수 있는 단위다.
- input/output/cache 비율이 어느 작업에서 커지는지 파악할 수 있다.

## 관찰 포인트
- tool 이름
- 호출 횟수
- 성공/실패
- 토큰 사용
- latency
- 관련 파일 활동
