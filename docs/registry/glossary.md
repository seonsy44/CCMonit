# CCMonit 용어집

- **canonical event**: 외부 로그를 CCMonit 내부 표준 이벤트로 정규화한 것
- **read model**: TUI와 리포트가 읽기 쉬운 형태로 projector가 계산한 상태
- **accuracy**: 수치나 관계가 얼마나 직접적/정확한지 나타내는 상태 (`exact`, `derived`, `estimated`, `unavailable`)
- **source page**: raw source를 위키에 편입하기 위한 요약 페이지
- **synthesis**: 여러 문서를 종합해 얻은 결론/관점 페이지
- **orphan event**: 상위 관계를 확정하지 못한 이벤트
- **stuck detection**: 장시간 유의미한 진행이 없는 작업을 탐지하는 규칙
- **docs steward**: 문서 구조와 위생을 책임지는 역할
