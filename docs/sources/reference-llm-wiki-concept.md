# LLM Wiki 컨셉 원문 참조 메모

이 파일은 첨부된 컨셉 문서를 바탕으로 `docs/` 시스템을 어떻게 재해석했는지 간단히 남긴 메모다.

- raw source는 원형 유지
- wiki는 증분 갱신
- schema는 LLM 유지 규칙
- query 결과도 위키에 축적
- index와 log를 별도 운영
