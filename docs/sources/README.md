# Source 페이지 가이드

`sources/`는 raw 문서 자체를 저장하는 공간이 아니라, **raw를 위키에 편입하기 위한 요약/등록 페이지**를 두는 공간이다.

## 역할

- raw origin 추적
- 핵심 포인트 요약
- 어느 문서가 영향을 받는지 기록
- 신규 페이지 필요 여부 판단
- 추후 lint 시 출처 역추적

## 원칙

- source page는 짧고 명확해야 한다
- source 하나가 여러 문서를 갱신하는 것이 정상이다
- raw 원본이 별도 위치에 있으면 링크나 경로만 남겨도 된다
