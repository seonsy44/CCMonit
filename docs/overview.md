# ccmonit Overview

## 제품 개요

ccmonit은 Claude Code 기반 개발 환경에서 **세션당 현재 무엇이 일어나고 있는가**를 실시간으로 모니터링할 수 있는 TUI 프로그램이다.

## 설치 및 실행

- 설치: `npm install @ccmonit/cli -g`
- 실행: 터미널에서 `ccmonit` 입력 시 메인 대시보드가 열린다.

## MVP 범위

### 포함

- **메인 대시보드**: 활성 세션 정보, 토큰 사용량, 도구 사용 현황, 스킬 사용 현황 (4개 섹션)
- **세션 목록 화면**: 종료된 세션 조회 및 삭제
- **세션 별칭**: 활성 세션에 사용자 지정 별칭 부여
- **키보드 단축키 기반 조작**

### 제외 (추후 구현 예정)

- 세션 레포트 (세션 목록 화면의 `r` 단축키)
- TUI 내 설정 화면
- 섹션 순서 변경 및 숨김 설정
- 도구 목록 정렬 기준 변경

## 화면 구성

| 화면 | 진입 방법 | 설명 |
|------|-----------|------|
| [메인 대시보드](./screens/main-dashboard.md) | `ccmonit` 실행 | 활성 세션 모니터링 |
| [세션 목록](./screens/session-list.md) | 대시보드에서 `l` 키 | 종료된 세션 조회/삭제 |

## 확장 계획

- 메인 대시보드의 섹션은 현재 4개이며 추후 추가될 수 있다.
- 세션 목록 외에 다른 화면도 추가될 수 있다.
- 섹션 순서 변경, 숨김 처리, 도구 정렬 기준 등 사용자 커스터마이징 기능이 추가될 예정이다.
- TUI 내 설정 화면이 추후 제공될 예정이다.
