# 질의 결과 보관 가이드

이 폴더는 “질문에 답하다가 나온 가치 있는 결과”를 저장하는 공간이다.

## 이 폴더에 넣는 것
- 여러 문서를 종합한 비교 결과
- 구현 방향에 영향을 주는 분석
- 토큰/비용/운영 패턴 해석
- 후속 논의를 부를 만한 요약

## 이 폴더에만 두면 안 되는 것
- 이미 장기 기준이 된 결론  
  → `knowledge/syntheses/` 또는 관련 spec 문서로 승격
- 의사결정 자체  
  → `decisions/`로 승격
