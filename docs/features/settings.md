# 기능: 설정 (Settings)

## 요약

ccmonit의 동작을 사용자가 커스터마이징할 수 있는 설정 항목을 정의한다.

## MVP 설정 제공 방식

- MVP에서는 TUI 내 설정 화면을 제공하지 않는다.
- 설정 파일 또는 CLI 플래그 등 별도 방식으로 제공한다.

## 추후 설정 제공 방식

- TUI 내 설정 화면을 추가하여 사용자가 대시보드에서 직접 설정을 변경할 수 있도록 한다.
- TUI 설정 화면 추가를 고려한 구조로 개발해야 한다.

## 설정 가능 항목 카탈로그

### 갱신 (Refresh)

| 항목                         | 설명                       | 디폴트 | MVP |
| ---------------------------- | -------------------------- | ------ | --- |
| token-usage-refresh-interval | 토큰 사용량 갱신 주기 (초) | 3      | O   |

### 정렬 (Sorting)

| 항목            | 설명                | 디폴트       | MVP |
| --------------- | ------------------- | ------------ | --- |
| tool-sort-order | 도구 목록 정렬 기준 | alphabetical | X   |

가능한 값: `alphabetical`, `count-desc`, `usage-order` (추후 정의)

### 섹션 (Sections)

| 항목           | 설명           | 디폴트                             | MVP |
| -------------- | -------------- | ---------------------------------- | --- |
| section-order  | 섹션 표시 순서 | [info, token-usage, tools, skills] | X   |
| section-hidden | 숨길 섹션 목록 | []                                 | X   |

### 참고

- MVP 열의 O는 MVP에서 설정 변경을 지원하는 항목, X는 추후 지원 예정인 항목이다.
- 추후 항목이 추가될 수 있다.
