# 기능: 토큰 사용량 표시 (Token Usage Display)

## 요약

사용자의 Claude Code 계정에 대한 토큰 사용량을 실시간으로 표시한다. Claude Code의 `/usage` 스킬이 제공하는 정보와 동일하다.

## 데이터 범위

- **계정 레벨** 데이터이다. 특정 세션과 무관하다.
- 세션을 전환해도 Token Usage 섹션의 내용은 변하지 않는다.

## 표시 항목

### Current Session (현재 세션 사용량)

- 진행률 바 (게이지)
- 사용 비율 (%)
- 리셋 시각

### Current Week (주간 사용량, 전체 모델 합산)

- 진행률 바 (게이지)
- 사용 비율 (%)
- 리셋 날짜 및 시각

## 갱신

- 주기적으로 자동 갱신된다.
- 갱신 주기는 사용자가 설정할 수 있다. (디폴트: 3초)
- 설정 항목: [settings.md](./settings.md) 참조

## 표시 위치

- 메인 대시보드의 section#1에 위치한다.
- 세션 목록 화면에서는 표시하지 않는다.
