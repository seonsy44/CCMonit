# 기능: 세션 생명주기 (Session Lifecycle)

## 요약

Claude Code 세션의 시작과 종료를 감지하고 상태를 추적한다.

## 세션 상태

| 상태 | 설명 |
|------|------|
| active | 현재 실행 중인 Claude Code 세션 |
| terminated | 종료된 세션 |

## 세션 시작 감지

- Claude Code가 새로 실행되면 활성 세션으로 등록한다.
- `/clear` 또는 `/compact` 사용 시 기존 세션은 종료되고, 새 세션이 시작된다.

## 세션 종료 감지

다음 조건 중 하나가 충족되면 세션을 종료로 판단한다:

- `/clear` 사용
- `/compact` 사용
- 터미널 닫기
- `exit` 명령으로 Claude Code 종료

## 세션 식별

- 각 세션은 고유한 세션 ID를 가진다 (예: `a81f8faa`).
- 세션 ID는 자동 생성된다.

## 세션 정보

활성 세션이 보유하는 정보:

| 필드 | 설명 | 변동성 |
|------|------|--------|
| session ID | 고유 식별자 | 불변 |
| model | 사용 중인 Claude 모델 | 실시간 변동 (`/model`로 변경 시 반영) |
| cwd | 작업 디렉토리 | 불변 (세션 중 변경 없다고 가정) |
| alias | 사용자 지정 별칭 | 사용자 변경 가능 |
| 시작 시각 | 세션 시작 시점 | 불변 |

종료 세션이 보유하는 정보:

| 필드 | 설명 |
|------|------|
| session ID | 고유 식별자 |
| cwd | 작업 디렉토리 |
| alias | 별칭 (변경 불가) |
| 시작 시각 | 세션 시작 시점 |
| 종료 시각 | 세션 종료 시점 |

## 복수 세션

- 여러 터미널에서 Claude Code를 동시에 실행하면 복수의 활성 세션이 존재한다.
- 메인 대시보드에서 ↑↓ 방향키로 활성 세션 간 전환할 수 있다.

## 빈 상태

- 활성 세션이 하나도 없는 경우 `(no running session)` 메시지를 표시한다.
