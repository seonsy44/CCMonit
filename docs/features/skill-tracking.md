# 기능: 스킬 사용 추적 (Skill Tracking)

## 요약

세션 내에서 사용된 스킬(슬래시 커맨드)의 목록과 상태를 추적하여 표시한다.

## 동작

- 사용자 또는 Claude AI가 스킬을 호출하면 목록에 추가된다.
- 동일 스킬을 여러 번 호출하면 각각 별도 항목으로 기록된다.
- 세션이 전환되면 해당 세션의 스킬 목록으로 교체된다.

## 표시 항목

각 스킬 항목은 다음 정보를 포함한다:

| 필드 | 설명 |
|------|------|
| 상태 아이콘 | 사용중(✓) 또는 완료(없음) |
| 스킬 이름 | 슬래시 커맨드명 (예: `/dev-sprint`) |
| 인자 | 스킬 호출 시 전달된 인자 (있는 경우) |
| 시각 | 호출 시각 (HH:MM:SS) |

## 스킬 상태

| 상태 | 표시 | 설명 |
|------|------|------|
| 사용중 | ✓ + 기본 밝기 | Claude가 해당 스킬의 작업을 수행 중 |
| 완료 | 흐림 처리 | Claude가 해당 스킬 사용을 끝냄 |

## 정렬

- 최신 호출이 위쪽에 위치한다 (역시간순).

## 표시 예시

```
✓ /dev-sprint (develop tui)        at 19:48:33
  /dev-sprint (develop service)    at 19:40:48
  /dev-open                        at 19:24:01
```

## 표시 위치

- 메인 대시보드의 section#3
- 세션 목록 화면의 section#2 (종료된 세션의 스킬 기록, 상태 구분 없이 모두 동일하게 표시)
