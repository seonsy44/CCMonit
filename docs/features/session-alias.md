# 기능: 세션 별칭 (Session Alias)

## 요약

사용자가 세션을 쉽게 식별할 수 있도록 별칭을 부여할 수 있다.

## 동작

- 메인 대시보드에서 `a` 단축키를 통해 현재 선택된 활성 세션에 별칭을 설정한다.
- 별칭은 영속적으로 저장된다. ccmonit을 종료했다가 다시 실행해도 유지된다.
- 세션이 종료된 후에도 별칭은 유지되며, 세션 목록 화면에서 확인할 수 있다.

## 제약

- **활성 세션에서만** 별칭을 설정/변경할 수 있다.
- 종료된 세션의 별칭은 변경할 수 없다 (읽기 전용).

## 표시

- 메인 대시보드의 Info 섹션(section#0)에 `alias:` 필드로 표시된다.
- 세션 목록 화면의 Info 섹션에도 동일하게 표시된다.
- 별칭이 설정되지 않은 세션은 해당 필드를 표시하지 않거나 빈 값으로 표시한다.
