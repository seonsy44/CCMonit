# 기능: 도구 사용 추적 (Tool Tracking)

## 요약

Claude Code가 세션 내에서 사용한 도구(Tool)의 종류와 사용 횟수를 추적하여 표시한다.

## 동작

- 세션 내에서 Claude Code가 도구를 사용할 때마다 해당 도구의 사용 횟수가 누적된다.
- 사용 횟수는 세션 단위로 집계된다.
- 세션이 전환되면 해당 세션의 도구 사용 현황으로 교체된다.

## 표시 형식

```
도구이름 횟수  도구이름 횟수  도구이름 횟수
```

예시:
```
Bash 41  Edit 11  Read 10  AskUserQuestion 5  Agent 4  Skill 2  Grep 2
ToolSearch 1  Write 2
```

- 도구 이름과 횟수를 쌍으로 표시한다.
- 터미널 너비에 맞춰 줄바꿈한다.

## 정렬

- 디폴트: 알파벳순
- 추후 사용자가 정렬 기준을 변경할 수 있도록 할 예정 (예: 사용 횟수 내림차순, 사용 순서 등)
- 설정 항목: [settings.md](./settings.md) 참조

## 표시 위치

- 메인 대시보드의 section#2
- 세션 목록 화면의 section#1 (종료된 세션의 최종 도구 사용 현황)
