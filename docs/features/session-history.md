# 기능: 세션 히스토리 (Session History)

## 요약

종료된 세션의 데이터를 영속적으로 보존하고, 사용자가 조회 및 삭제할 수 있다.

## 영속성

- 종료된 세션의 데이터(세션 정보, 도구 사용 현황, 스킬 기록)는 영속적으로 저장된다.
- ccmonit을 종료했다가 다시 실행해도 종료된 세션 목록은 유지된다.
- 사용자가 명시적으로 삭제하지 않는 한 데이터는 보존된다.

## 조회

- 세션 목록 화면(메인 대시보드에서 `l` 키)에서 종료된 세션을 조회할 수 있다.
- 종료된 세션이 여러 개인 경우 ↑↓ 방향키로 전환하며 상세 정보를 확인한다.
- 정렬: 시작 시각 순서로 나열된다.

## 삭제

- 세션 목록 화면에서 `d` 단축키로 현재 선택된 종료 세션을 삭제할 수 있다.
- 삭제 전에 사용자에게 확인을 요청한다 ("정말 삭제하시겠습니까?" 등).
- 삭제 시 해당 세션의 모든 데이터(세션 정보, 도구 사용 현황, 스킬 기록, 별칭)가 **완전히 삭제**된다.
- 삭제된 데이터는 복구할 수 없다.

## 종료 세션이 보유하는 데이터

- 세션 정보: session ID, cwd, alias, 시작 시각, 종료 시각
- 도구 사용 현황: 도구별 최종 사용 횟수
- 스킬 기록: 호출된 스킬 목록 (이름, 인자, 시각)

## 레포트 (추후 구현)

- 세션 목록 화면에서 `r` 단축키로 선택된 종료 세션에 대한 레포트를 볼 수 있다.
- MVP에서는 구현하지 않으며, 추후 구현 예정이다.
