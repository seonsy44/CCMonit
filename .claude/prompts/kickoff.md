# kickoff prompt

이번 세션의 새 작업을 정하기 위한 시작 템플릿이다.

## 먼저 읽을 것
- `CLAUDE.md`
- `docs/operations/claude-code-session-workflow.md`
- `.claude/context/product-summary.md`
- `.claude/context/architecture-summary.md`
- `.claude/context/docs-system-summary.md`
- `.claude/context/glossary.md`
- 필요 시 `docs/index.md`, `docs/log.md`, `docs/registry/page-index.md`, `.claude/scratch/*`

## 이번 세션 목표
- 이번 세션에서 지금 바로 할 수 있는 작은 작업 1~3개를 추천한다.
- 각 작업의 범위와 이유를 짧게 적는다.
- 가장 추천하는 작업 1개를 고른 뒤, 다음 단계로 `/session-start`에서 무엇을 확인해야 하는지 정리한다.

## 작업 원칙
- docs를 source of truth로 본다.
- 작은 배치로 작업한다.
- 완료 후 `.claude/scratch/worklog.md` 와 `next-prompt.md`를 갱신할 수 있게 정리한다.
- 관련 docs 변경이 필요하면 같이 표시한다.

## 원하는 출력
1. 지금 가능한 작은 작업 후보 1~3개
2. 가장 추천하는 작업 1개
3. 읽어야 할 파일 목록
4. `/session-start`에서 고정할 범위
5. 종료 시 업데이트할 문서/메모 목록
