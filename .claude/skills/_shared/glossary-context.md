# Glossary Context for Skills

- session: 하나의 모니터링 대상 작업 흐름 단위
- batch: 저비용 세션에서 한 번에 처리하는 아주 작은 구현 단위
- stop line: 이번 세션에서 여기까지만 하고 멈춘다는 경계
- canonical event: 어댑터별 raw 로그를 공통 포맷으로 정규화한 이벤트
- token accuracy: exact / derived / estimated / unavailable
- read model: TUI나 리포트가 바로 소비하기 좋은 형태의 요약 모델
- adapter health: 로그 감지기 / 파서 / 토큰 추출기의 신뢰도와 정상 상태
- docs ingest: 새 사실을 위키 문서에 편입하는 작업
- docs lint: 위키의 누락 / 충돌 / 고아 문서를 점검하는 작업
