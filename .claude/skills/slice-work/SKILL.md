작업 목표: **$ARGUMENTS**

## 목적
큰 목표를 20~40분 내 처리 가능한 배치로 분해한다.

## 규칙
- 배치 하나는 가능한 한 하나의 책임만 가진다.
- 배치 하나는 1~4개 파일 수정 수준으로 제한한다.
- 각 배치에는 명확한 완료 조건이 있어야 한다.
- 지금 바로 시작 가능한 배치를 최우선으로 선정한다.

## 출력 요구
다음을 포함하라.
1. 전체 목표 요약
2. 배치 목록 표
3. 각 배치의 선행 조건
4. 가장 먼저 할 배치 1개
5. 그 배치에 대해 사용할 구현 스킬 추천

## 표 형식
| Batch ID | 목적 | 수정 예상 파일 | 완료 조건 | 선행 조건 |
